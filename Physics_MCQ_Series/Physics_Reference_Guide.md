# Physics Reference Guide: Signs, Symbols & Formulas (Premium Series)

This comprehensive guide is designed for students, educators, and healthcare professionals. It covers the essential mathematical language of physics, including the Greek alphabet, trigonometric values, and a master list of formulas across all major physics domains.

## Part 1: Mathematical Signs & Operational Symbols

| Symbol | Name | Usage in Physics |
| :--- | :--- | :--- |
| $\Delta$ | Delta | Change in a quantity (e.g., $\Delta v$ for change in velocity) |
| $\sum$ | Sigma | Summation (e.g., $\sum F$ for net force) |
| $\propto$ | Proportional | Shows relationship between variables ($a \propto F$) |
| $\approx$ | Approximation | Used for estimated values ($\pi \approx 3.14159$) |
| $\nabla$ | Del / Nabla | Vector differential operator (Gradient/Divergence) |
| $\infty$ | Infinity | Used for limits or object distance at infinity |
| $|x|$ | Absolute Value | Magnitude of a vector or scalar (eliminating sign) |
| $\langle x \rangle$ | Average | Denotes the mean value of a variable |
| $\sqrt{}$ | Square Root | Used in RMS values and many velocity formulas |
| $\partial$ | Partial Derivative | Used in advanced thermodynamics and wave equations |
| $\int$ | Integral | Calculating area under curves (Work, Displacement) |

## Part 2: The Greek Alphabet in Physics

Many Greek letters are used to represent specific constants and variables.

| Letter | Symbol | Common Physics Usage |
| :--- | :--- | :--- |
| **Alpha** | $\alpha$ | Angular acceleration, Alpha particles, Temperature coefficient |
| **Beta** | $\beta$ | Beta particles, Velocity/c ratio (Special Relativity) |
| **Gamma** | $\gamma$ | Gamma radiation, Lorentz factor, Specific heat ratio |
| **Delta** | $\delta$ | Small change, Phase constant |
| **Epsilon** | $\epsilon$ | Permittivity of vacuum, Strain, Electromotive force (EMF) |
| **Theta** | $\theta$ | Angle, Temperature (sometimes) |
| **Lambda** | $\lambda$ | Wavelength, Linear charge density, Decay constant |
| **Mu** | $\mu$ | Coefficient of friction, Micro ($10^{-6}$), Permeability |
| **Nu** | $\nu$ | Frequency of a wave (often used interchangeably with $f$) |
| **Pi** | $\pi$ | Circle constant, $3.14159...$ |
| **Rho** | $\rho$ | Density, Resistivity |
| **Sigma** | $\sigma$ | Conductivity, Surface tension, Stefan-Boltzmann constant |
| **Tau** | $\tau$ | Torque, Time constant |
| **Phi** | $\phi$ | Magnetic flux, Work function |
| **Psi** | $\psi$ | Wave function (Quantum Mechanics) |
| **Omega** | $\omega, \Omega$ | Angular velocity ($\omega$), Ohm (Resistance, $\Omega$) |

## Part 4: The Physics Formula Master List

### üõ∞Ô∏è Mechanics & Motion
*   **Average Speed:** $v = s / t$
*   **Uniform Acceleration:** $a = (v_f - v_i) / t$
*   **Kinematic equations:**
    1. $v_f = v_i + at$
    2. $s = v_i t + \frac{1}{2} at^2$
    3. $v_f^2 = v_i^2 + 2as$
*   **Force:** $F = ma$
*   **Momentum:** $p = mv$
*   **Newton's Gravitation:** $F = G \frac{M m}{r^2}$

### ‚ö° Electricity & Magnetism
*   **Net Charge:** $Q = It$ (Current $\times$ Time)
*   **Ohm's Law:** $V = IR$
*   **Electric Power:** $P = VI = I^2 R = V^2 / R$
*   **Electrical Energy:** $E = Pt = VIt$ (Commercial unit: $1 \text{ kWh} = 3.6 \times 10^6 \text{ J}$)
*   **Resistance (Series):** $R_T = R_1 + R_2 + R_3...$
*   **Resistance (Parallel):** $\frac{1}{R_T} = \frac{1}{R_1} + \frac{1}{R_2} + \frac{1}{R_3}...$
*   **Magnetic Force:** $F = qvB \sin \theta$ (on a moving charge)

### üå°Ô∏è Thermal & Matter
*   **Density:** $\rho = \frac{m}{V}$
*   **Pressure:** $P = \frac{F}{A}$
*   **Pressure in Liquids:** $P = \rho gh$ (Height $\times$ Density $\times$ Gravity)
*   **Thermal Expansion:** $\Delta L = \alpha L_o \Delta T$
*   **Specific Heat:** $Q = mc \Delta T$
*   **Ideal Gas Law:** $PV = nRT$

### üåä Waves & Optics
*   **Wave Speed:** $v = f \lambda$
*   **Frequency:** $f = \frac{1}{T}$
*   **Refractive Index:** $n = \frac{c}{v}$ (Speed of light in vacuum / speed in medium)
*   **Snell's Law:** $n_1 \sin \theta_1 = n_2 \sin \theta_2$
*   **Lens Power:** $P = \frac{1}{f}$ (where $f$ is in meters; Unit: Dioptre)
*   **Magnification:** $m = \frac{h_i}{h_o} = -\frac{v}{u}$

### üß™ Atomic & Modern Physics
*   **Photon Energy:** $E = hf = \frac{hc}{\lambda}$
*   **Mass-Energy:** $E = mc^2$
*   **Photoelectric Effect:** $hf = \text{Work Function} + K.E._{max}$
*   **Radioactive Decay:** $N_t = N_o e^{-\lambda t}$
*   **Einstein's Relativity (Gamma):** $\gamma = \frac{1}{\sqrt{1 - v^2/c^2}}$

## Part 5: Physics Constants to Remember

*   **Acceleration due to gravity ($g$):** $9.81 m/s^2$ (approx. $10 m/s^2$)
*   **Speed of light ($c$):** $3 \times 10^8 m/s$
*   **Planck's Constant ($h$):** $6.63 \times 10^{-34} J \cdot s$
*   **Universal Gas Constant ($R$):** $8.314 J/(mol \cdot K)$
*   **Permittivity of Free Space ($\epsilon_0$):** $8.85 \times 10^{-12} F/m$
*   **Coulomb's Constant ($k$):** $8.98 \times 10^9 N \cdot m^2/C^2$
